Project: Wikipedia Viewer for Free Code Camp

My attempt at the Wikipedia Viewer project for FCC.

This app allows the user to search for articles from Wikipedia by using the Wikipedia API. The user also has the option to visit a random Wikipedia article by clicking on the 'Random' button.

Technologies Used:

  • Bootstrap
  • jQuery
  • Wikipedia API

Future Features:

  • Change the styling of the results and format differently (possibly as a grid?).
